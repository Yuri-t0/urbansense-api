Endpoints Unificados

Todos os microsserviços acessíveis por uma única entrada via Spring Cloud Gateway.

GET/api/v1/weather/currentClima atual de uma cidade
GET/api/v1/weather/rain-riskAnálise histórica de risco
GET/api/v1/weather/heavy-rainRegiões com chuva intensa
POST/api/v1/alerts/evaluateAvaliação de alertas urbanos
GET/api/v1/alerts/city/{city}Alertas ativos por cidade
GET/api/v1/alerts/type/{type}Alertas filtrados por tipo
GET/api/v1/mobility/lines-at-riskLinhas de metrô em risco
POST/api/v1/advisor/askConsulta ao assistente IA
GET/api/v1/advisor/daily-summary/{city}Resumo diário com score
GET/api/v1/mcp/toolsCatálogo MCP para LLMs externos
POST/api/v1/auth/loginAutenticação JWT
